## 注意事项
如果这几种标识符是下面的保留字作为数据库名、表名、列名、函数名、视图名等标识符的时候，需要加上默认的转义符：
```
标识符：` 反引号

例如: hour -> `hour`

create table `hour` (
id string,
`asc` int
)

hour、asc 为关键字，此时如果需要创建这样的列就必须带上反引号进行创建。
```
